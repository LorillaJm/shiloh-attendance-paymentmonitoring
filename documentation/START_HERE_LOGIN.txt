================================================================================
                        üîß TROUBLESHOOTING
================================================================================

Problem: Can't access the login page
Solution:
    1. Make sure server is running: php artisan serve
    2. Check URL: http://localhost:8000/admin/login
    3. Try: http://127.0.0.1:8000/admin/login

Problem: "These credentials do not match our records"
Solution:
    1. Double-check email: admin@admin.com (no spaces)
    2. Double-check password: admin (lowercase)
    3. Try the other account: admin@shiloh.local / password
    4. Recreate user: php artisan db:seed --class=SimpleAdminSeeder

Problem: Server won't start
Solution:
    1. Check if port 8000 is in use
    2. Try different port: php artisan serve --port=8001
    3. Check database is running

Problem: Database error
Solution:
    1. Check .env file has correct database settings
    2. Run: php artisan migrate
    3. Check PostgreSQL is running
